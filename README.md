## What It Is

A constitution-governed multi-agent think tank that assembles a 
team of domain specialists to analyze complex engineering problems 
and generate ranked solution architectures. Originally developed 
for lunar colony requirements analysis at Shackleton Crater.

Five specialist agents run in parallel, each grounded in domain-
specific RAG cabinet data, governed by a shared physical law 
constitution. A Mission Director synthesizes across three debate 
rounds to produce ranked, feasibility-scored solutions.

All inference runs locally via Ollama. No cloud dependencies.
